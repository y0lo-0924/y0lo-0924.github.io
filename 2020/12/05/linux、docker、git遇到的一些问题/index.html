linux、docker、git遇到的一些问题

最近在使用kali、docker、git的过程中遇到了一下问题总结下来。

0x01 kali换源问题

  • vim /etc/apt/sources.list

  • 加入源文件

  • 更新一下源：

    apt-get update

  • 更新软件：

    apt-get upgrade

  • 更新系统：

    apt-get dist-upgrade

  • 清理安装包：

    apt-get clean

    apt-get autoclean

0x02 docker更换国内源

更改镜像源，系统没有这个文件，直接新增这个文件

  • vim /etc/docker/daemon.json

  • 加入源文件

  • 更改镜像后重启docker

    systemctl restart docker

  • 查看docker是否运行成功

    pd -ef |grep docekr

  • docker查看本地镜像

    docker image ls

  • docker下载，默认是下载latest

    docker pull nginx

  • 下载指定版本

    docker pull nginx:1.12

  • 删除镜像

    docker image rm <id|name>

0x03 进入docker容器

docker ps查看dockerip

docker exec -it 【dockr-id】 bash
